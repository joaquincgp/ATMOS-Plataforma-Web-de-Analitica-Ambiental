# Azure setup notes

Este directorio contiene convenciones de despliegue para Azure App Service.

Pasos recomendados:

1. Crear Resource Group y App Service Plan Linux.
2. Crear web app para backend (Python 3.11).
3. Crear web app para frontend (Node 20) o static hosting.
4. Crear PostgreSQL Flexible Server y habilitar extension PostGIS.
5. Configurar GitHub Secrets para workflows de deploy.
